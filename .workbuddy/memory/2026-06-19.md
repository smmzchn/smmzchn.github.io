# 2026-06-19 工作日志

## 搭建个人博客

为时笠搭建了暗色调静态个人博客，包含：

- `index.html` — 首页（文章列表 + 关于区域）
- `post.html` — 文章渲染页（通过 URL 参数加载 Markdown）
- `style.css` — 暗色主题（#0d1117 背景，#00d4aa 强调色）
- `script.js` — 文章列表渲染逻辑（POSTS 数组维护）
- `posts/hello-world.md` — 生活类示例文章（博客开张随笔）
- `posts/js-closure.md` — 技术类示例文章（JavaScript 闭包讲解）

技术要点：
- Markdown 渲染用 marked.js CDN
- 代码高亮用 highlight.js CDN
- 文章元数据用 frontmatter（--- 包裹）
- 新增文章只需在 `script.js` 的 POSTS 数组加一条记录
