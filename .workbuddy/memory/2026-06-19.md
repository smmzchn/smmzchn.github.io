# 2026-06-19 工作日志

## 搭建个人博客

为时笠搭建了暗色调静态个人博客，包含：

- `index.html` — 首页（文章列表 + 关于区域）
- `post.html` — 文章渲染页（通过 URL 参数加载 Markdown）
- `style.css` — 暗色主题（#0d1117 背景，#00d4aa 强调色）
- `script.js` — 文章列表渲染逻辑（POSTS 数组维护）
- `posts/hello-world.md` — 生活类示例文章（博客开张随笔）
- `posts/js-closure.md` — 技术类示例文章（JavaScript 闭包讲解）

技术要点：
- Markdown 渲染用 marked.js CDN
- 代码高亮用 highlight.js CDN
- 文章元数据用 frontmatter（--- 包裹）
- 新增文章只需在 `script.js` 的 POSTS 数组加一条记录

## 添加图片上传和预览图功能

为博客添加了图片上传和预览图功能：

- `admin.html` — 添加了图片上传按钮和预览图上传功能
  - 预览图：上传到 `assets/images/` 目录，URL 存进文章的 frontmatter（`previewImage` 字段）
  - 文章图片：选择图片后自动上传到 GitHub，插入 `![图片](url)` 到文章里
- `post.html` — 文章页面现在会显示预览图（`previewImage` 字段）
- `script.js` — 文章列表现在会显示预览图（如果有）
- `build_posts.py` — 新增的脚本，用来重新生成 `posts.json`（把文章内容嵌进 JSON，彻底绕开 GitHub Pages 不提供 `.md` 文件的问题）
- `posts.json` — 更新文章数据，包含 `previewImage` 和 `content` 字段

技术要点：
- 图片上传通过 GitHub API（`PUT /repos/{owner}/{repo}/contents/{path}`）
- 图片文件转 base64 后上传
- 返回 `raw.githubusercontent.com` 的 URL
